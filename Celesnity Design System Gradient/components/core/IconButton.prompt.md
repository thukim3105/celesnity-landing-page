**IconButton** — a compact single-icon control for toolbars, nav bars, and card corners. Always pass `ariaLabel`.

```jsx
<IconButton ariaLabel="Settings" variant="ghost"><GearIcon/></IconButton>
<IconButton ariaLabel="Add" variant="primary" shape="rounded"><PlusIcon/></IconButton>
```

Mirrors Button's variants; `circle` (default) or `rounded`.
